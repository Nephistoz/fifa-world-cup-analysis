# FIFA World Cup Analysis

A data science project analyzing FIFA World Cup team performance and predicting match outcomes.

## Project Overview

This project aims to:
- **Exploratory Data Analysis (EDA)**: Analyze historical team performance metrics
- **Predictions**: Build models to predict match outcomes

## Project Structure

```
fifa-world-cup-analysis/
├── data/
│   ├── raw/                    # Original, immutable data
│   └── processed/              # Cleaned and transformed data
├── notebooks/
│   ├── 01_eda.ipynb           # Exploratory data analysis
│   ├── 02_data_preprocessing.ipynb
│   └── 03_predictions.ipynb   # Prediction models
├── results/                    # Output, figures, predictions
├── requirements.txt            # Project dependencies
└── README.md
```

## Getting Started

1. Clone the repository
2. Create a virtual environment: `python -m venv venv`
3. Activate it and install dependencies: `pip install -r requirements.txt`
4. Start with notebooks in the `notebooks/` folder
